# Pokédex - Desafio Front-End

Aplicação desenvolvida como parte de um desafio técnico utilizando:

- React
- TypeScript
- Axios
- TailwindCSS
- PokeAPI

## Funcionalidades

- Listagem dos 151 primeiros Pokémon
- Filtro por nome (client-side)
- Página de detalhes dinâmica
- Tratamento de loading e erro
- Navegação com React Router

## Como rodar o projeto

```bash
npm install
npm run dev
